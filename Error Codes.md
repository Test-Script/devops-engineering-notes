==============================================================================================================
                                    Common Error Codes (4xx and 5xx)
==============================================================================================================

400 Bad Request: Indicates a malformed request.
401 Unauthorized: User authentication is required.
403 Forbidden: Access to the resource is denied.
404 Not Found: The requested page or resource cannot be found.
405 Method Not Allowed: The HTTP method used is not supported.
429 Too Many Requests: The user is experiencing rate limiting.
500 Internal Server Error: A general, unspecific server-side error.
502 Bad Gateway: Invalid response from an upstream server.
503 Service Unavailable: Server is overloaded or under maintenance.
504 Gateway Timeout: Upstream server failed to respond in time.
